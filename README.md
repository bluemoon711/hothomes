Description:
A housing-booking website like Airbnb. A visitor can search for available places but need to sign up as a user to make the reservation. 
Hot Homes users may also become landlords to post their properties on the website. After logging in Hot Homes, a user can view either his
bookings as a traveler and/or view his properties as a landlord.

Technologies:
Built with React and Redux, scaffolded on LoopBack and NodeJs, using MongoDB as database, styled with Bulma and LESS.
